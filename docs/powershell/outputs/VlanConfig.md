### VlanConfig
Supported in v5.0+

- interfaces: list of NodeIps
  - Required. Supported in v5.0+
  Network interfaces for each node.
- netmask: System.String
  - Required. Supported in v5.0+
  Netmask for addresses on this VLAN.
- vlan: System.Int32
  - Required. Supported in v5.0+
- alias: System.String
  - Alias for the VLAN.
